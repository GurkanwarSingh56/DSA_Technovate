# DSA Practice Repository

This repository is a beginner-friendly DSA practice space for Technovate students. It is designed to help you build problem-solving skills step by step, starting from the basics and gradually moving toward more challenging questions.

## Learning Path

We will divide the practice into 3 parts:

1. Beginner
   - Very basic DSA questions
   - Focus on understanding concepts
   - Learn how to write simple logic clearly

2. Intermediate
   - Questions that combine multiple concepts
   - Improve speed, accuracy, and pattern recognition
   - Practice with slightly harder problem-solving tasks

3. Upper Intermediate
   - More advanced questions
   - Strengthen algorithmic thinking
   - Prepare for stronger interview-style problems

## Goal

The main goal of this repo is to help Technovate students practice DSA in a structured way, without feeling overwhelmed. Each level is meant to build confidence before moving to the next.

## How to Use This Repo

- Start with the beginner section if you are new to DSA.
- Solve problems in order when possible.
- Focus on understanding the approach, not just the answer.
- Keep revising earlier topics while moving ahead.

## Contribution

Students can add solutions, notes, and better approaches as they progress. Keep the code clear, readable, and easy to understand for others.
